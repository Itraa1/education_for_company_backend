📦 ЧТО БЫЛО СОЗДАНО:

  1️⃣  Роль "Author" (Автор)
      • Может создавать курсы
      • Может редактировать свои курсы
      • Видит кто прошел их курсы
      • НЕ может удалять курсы

  2️⃣  Роль "Admin" (Администратор)
      • Полный доступ ко всему
      • Может управлять пользователями
      • Может управлять ролями
      • Может удалять любые курсы

  3️⃣  Модель "Course Completion" (Прохождение курса)
      • Отслеживает какой пользователь какой курс прошел
      • Хранит прогресс (0-100%)
      • Хранит дату завершения
      • Может содержать заметки пользователя

🔌 API ENDPOINTS:


  Создание курса (Author/Admin только):
  POST /api/courses
    + Authorization: Bearer TOKEN
    + data: { title, description, category, level, duration, instructor }


  Отслеживание прохождения:
  POST /api/course-completions
    + Authorization: Bearer TOKEN
    + data: { user, course, progress, completedAt, notes }


  Получить прохождения пользователя:
  GET /api/course-completions?filters[user][id][$eq]=1


  Получить кто прошел курс:
  GET /api/course-completions?filters[course][id][$eq]=1
