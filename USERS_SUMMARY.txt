## 👥 Роли пользователей в вашем Strapi

### 🔐 Сейчас установлены (по умолчанию):

1. **Public** 🌐 - Неавторизованные пользователи (только чтение)
2. **Authenticated** 🔑 - Авторизованные пользователи

---

### 🎯 Рекомендуемые роли для платформы обучения:

**Для создания в админ панели:**

| Роль | Назначение | Может делать |
|------|-----------|---------|
| **Student** 👨‍🎓 | Студент | Просматривать курсы, регистрироваться, видеть оценки |
| **Instructor** 👨‍🏫 | Преподаватель | Создавать/редактировать курсы, выставлять оценки |
| **Manager** 📋 | Менеджер контента | Модерировать контент, удалять курсы |
| **Admin** 🔧 | Администратор | Полный доступ, управление пользователями |

---

### 📊 Как работает система:

```
Пользователь
  ├─ Роль (одна или несколько)
  │   └─ Разрешения (create, read, update, delete)
  │       └─ На конкретную коллекцию (courses, lessons, etc)
  │
  └─ JWT Token (для API запросов)
```

---

### 🔗 API для пользователей:

```
POST   /api/auth/local/register    - Регистрация
POST   /api/auth/local             - Вход (получить JWT)
GET    /api/users/me               - Текущий пользователь
GET    /api/users                  - Все пользователи (Admin)
```

---

### 💾 В БД используются таблицы:

- `users_permissions_role` - Роли
- `users_permissions_permission` - Разрешения  
- `users_permissions_user` - Пользователи
- `users_permissions_users_role` - Связь пользователь-роль

---

### ⚙️ Где управлять ролями:

**Admin панель:** http://localhost:1337/admin
- Settings → Users & Permissions → Roles
