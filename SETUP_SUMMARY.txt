📊 СТРУКТУРА ДАННЫХ COURSE:

┌─ COURSE
├─ id (auto, integer)
├─ title * (string: 3-255 символов)
├─ slug * (uid: автогенерируется)
├─ description * (richtext: HTML/Markdown)
├─ category * (enum)
│  ├─ programming
│  ├─ design
│  ├─ business
│  ├─ marketing
│  ├─ management
│  ├─ soft-skills
│  └─ other
├─ level * (enum)
│  ├─ beginner
│  ├─ intermediate
│  └─ advanced
├─ duration * (integer: минуты)
├─ instructor * (string: 1-255 символов)
├─ createdAt (auto, timestamp)
├─ updatedAt (auto, timestamp)
└─ publishedAt (auto, timestamp)

* = обязательное поле

════════════════════════════════════════════════════════════════════════════════

🔌 API ENDPOINTS (после npm run setup-course):

  GET    /api/courses              # Получить все курсы
  GET    /api/courses/:id          # Получить курс по ID  
  POST   /api/courses              # Создать новый курс
  PUT    /api/courses/:id          # Обновить курс
  DELETE /api/courses/:id          # Удалить курс
  POST   /api/courses/:id/publish  # Опубликовать курс

Q: Как проверить что установилось?
A: ls -la src/api/course/    (Linux/Mac)

A: rm -rf src/api/course  &&  npm run setup-course
